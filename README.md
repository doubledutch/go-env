# Env

Read from the environment like you read from the command line with `flag`.
Env is designed to be a drop in replacement for `flag` package.

## Supported
* [x] String
* [x] StringVar
* [x] Int
* [x] IntVar
* [x] Bool
* [x] BoolVar
